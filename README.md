# 🧠 Sinalibras - Frontend

## 📌 Descrição
Projeto desenvolvido como Trabalho de Conclusão de Curso (TCC) no SENAI, com o objetivo de criar uma plataforma voltada para acessibilidade em Libras.

Fui responsável pelo desenvolvimento do front-end da aplicação, além de contribuir com a concepção geral do sistema.

---

## 💻 Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- (coloca React se tiver usado)

---

## ⚙️ Funcionalidades

- Interface de navegação para usuários
- Sistema de interação com conteúdos
- Integração com API (back-end)
- Layout responsivo

---

## 👨‍💻 Minha contribuição

- Desenvolvimento completo do front-end
- Criação das telas e layout
- Participação na idealização do projeto
- Colaboração com equipe (back-end e banco de dados)

---

## 📚 Aprendizados

- Desenvolvimento em equipe
- Integração entre front-end e back-end
- Organização de projeto
- Boas práticas de desenvolvimento web
